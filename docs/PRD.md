# 마이링크 (MyLink) - 제품 요구사항 정의서 (PRD)

## 1. 프로젝트 개요
- **프로젝트명**: 마이링크 (MyLink)
- **목적**: 여러 개의 링크를 하나의 통합된 프로필 페이지로 관리하고 공유할 수 있는 링크트리(Linktree) 클론 서비스. 사용자의 분산된 온라인 활동 내역을 한 곳으로 집중시킴.
- **대상 사용자**: 
  - 다양한 채널(유튜브, 인스타그램, 블로그 등)을 운영하는 크리에이터 및 인플루언서
  - 포트폴리오, 이력서, 개인 작업물 등을 간편하게 공유하고자 하는 취업준비생/개인
  - 다수의 링크(쇼핑몰, 이벤트 페이지, 소셜 미디어 등)를 하나로 묶어 고객에게 노출해야 하는 소상공인 및 브랜드

## 2. 핵심 기능 목록

### 필수 기능 (Must-Have)
1. **사용자 계정 관리 (회원가입/로그인)**
2. **링크 블록 관리 (CRUD 및 정렬)**
3. **프로필 관리 (이미지 및 소개글)**
4. **마이링크 페이지 퍼블리싱 (고유 URL 제공)**

### 선택 기능 (Nice-to-Have) 및 개선 제안 사항
1. **통계 대시보드 (방문자 수 및 클릭 수 분석)** - *제안*
2. **디자인 커스터마이징 (테마, 배경, 버튼 스타일)** - *제안*
3. **소셜 로그인 (카카오, 구글, 깃허브 등 간편 로그인)** - *제안*
4. **SEO 및 메타 태그 최적화 (Open Graph 공유 썸네일 노출)** - *제안*

## 3. 기능의 상세 설명

### [필수] 1. 사용자 계정 관리
- **회원가입 및 로그인**: 이메일과 비밀번호를 이용한 기본 인증 처리.
- **URL(아이디) 선점**: 가입 시 자신의 고유 링크로 사용할 아이디(`domain.com/username`) 설정 기능.
- **계정 설정**: 비밀번호 변경 및 회원 탈퇴 기능.

### [필수] 2. 링크 블록 관리
- **링크 추가/수정/삭제**: 사용자가 노출하고 싶은 웹주소(URL)와 제목(Title)을 입력하여 링크 버튼 생성.
- **드래그 앤 드롭 정렬**: 마우스 드래그를 통해 직관적으로 링크 버튼의 노출 순서를 변경.
- **숨기기 (토글)**: 필요에 따라 링크를 삭제하지 않고 임시로 화면에 보이지 않도록 ON/OFF 할 수 있는 스위치 기능.

### [필수] 3. 프로필 관리
- **프로필 이미지 설정**: 페이지 상단에 표시될 아바타 이미지 업로드 및 적용 기능.
- **이름 및 바이오(Bio)**: 닉네임과 자신을 설명하는 짧은 소개글 작성/수정 기능.

### [필수] 4. 마이링크 페이지 퍼블리싱
- **웹 페이지 제공**: 사용자 아이디를 기반으로 한 고유 주소로 접근 시, 추가한 링크와 프로필이 반영된 뷰어(Viewer) 페이지 노출.
- **반응형 웹**: 주로 인스타그램 프로필 등 모바일에서 접근하는 경우가 많으므로 모바일 화면에 완벽히 최적화된 UI 제공.
